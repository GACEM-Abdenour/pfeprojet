\chapter*{Résumé}
\addcontentsline{toc}{chapter}{Résumé}

Ce mémoire de fin d'études, réalisé au sein de la \textbf{Direction Informatique de Naftal,
Branche Carburants}, porte sur la conception et le développement d'une plateforme web
centralisée pour la \textbf{Gestion des Incidents Applicatifs (GIA)}. Naftal, filiale à 100\%
de Sonatrach, est le leader national algérien de la distribution des produits pétroliers et
opère un Système d'Information (SI) complexe sur l'ensemble du territoire. Face aux
limites du support informatique actuel — déclarations par téléphone, absence de
traçabilité, perte d'informations — notre solution digitalise et centralise ce processus.

La plateforme repose sur une architecture à trois niveaux (3-Tiers) : une interface
développée avec \textbf{Bootstrap 5}, une logique métier en \textbf{PHP 8}, et une base de
données \textbf{Microsoft SQL Server Express}. La solution est déployée sur
\textbf{Windows Server 2019} via IIS, conformément aux exigences d'infrastructure de
l'entreprise, simulée sur une machine virtuelle VirtualBox.

\vspace{0.5cm}
\noindent\textbf{Mots-clés :} ITSM, Helpdesk, Ticketing, PHP, SQL Server, Windows Server 2019, IIS, Naftal, Bootstrap, Système d'Information.

\vspace{2cm}
\begin{center} \Large \textbf{Abstract} \end{center}

This graduation project, carried out within the IT Directorate of \textbf{Naftal, Fuel
Branch}, focuses on the design and development of a centralized web platform for
\textbf{Application Incident Management (GIA)}. Naftal, a 100\% subsidiary of
Sonatrach, is Algeria's national leader in petroleum product distribution and relies on a
complex Information System. Facing the limitations of the current IT support process —
phone-based reporting, lack of traceability, information loss — our solution fully
digitalizes and centralizes this workflow.

The platform is built on a 3-Tier architecture: a \textbf{Bootstrap 5} frontend, a
\textbf{PHP 8} business logic layer, and a \textbf{Microsoft SQL Server Express} database.
It is deployed on \textbf{Windows Server 2019} using IIS with FastCGI, matching Naftal's
corporate infrastructure standards, tested on a VirtualBox virtual machine.
